# AI Resume Analyzer & Job Matching System 🚀

This project is an AI-powered application that analyzes resumes and matches them with job descriptions.

## 🔍 Features
- Extracts text from PDF resumes
- Identifies key skills using NLP
- Matches resume with job description
- Calculates match score (%)
- Shows matched & missing skills
- Clean UI with Streamlit

## 🛠 Tech Stack
- Python
- Streamlit
- NLP (Text Processing)
- PDF Parsing

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python -m streamlit run app.py
